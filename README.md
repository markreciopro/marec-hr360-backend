HR360 Backend — FastAPI + Supabase + Render
HR360 is a modular HR analytics backend built with FastAPI, deployed on Render, and powered by Supabase PostgreSQL.
It processes CSV uploads, cleans and transforms HR datasets, and generates analytics for dashboards and reporting.

🚀 Features
CSV upload endpoint

Data cleaning pipeline (clean_data)

Data transformation pipeline (transform_data)

Analytics engine (compute_analytics)

Local + cloud database switching

Production‑ready Render deployment

Modular folder structure for scalability
